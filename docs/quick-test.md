# ⚡ 빠른 기능 테스트 (5분)

## 1️⃣ 페이지 로드 (30초)
- [ ] 페이지가 열림
- [ ] 4개 메트릭 카드 표시됨
- [ ] 로그 목록 표시됨
- [ ] 콘솔 에러 없음

## 2️⃣ 메트릭 확인 (30초)
- [ ] CPU 값 표시됨: _____%
- [ ] 메모리 값 표시됨: _____ MB / _____ MB
- [ ] 디스크 값 표시됨: _____ GB / _____ GB
- [ ] 네트워크 값 표시됨: IN _____ / OUT _____ Mbps

## 3️⃣ 자동 갱신 (1분)
- [ ] 5초 대기
- [ ] "마지막 업데이트" 시간 변경됨
- [ ] 메트릭 값이 변경됨
- [ ] 콘솔에 "메트릭 업데이트 완료" 로그

## 4️⃣ 로그 검색 (1분)
1. 검색창에 "데이터베이스" 입력
2. 0.3초 대기
- [ ] 매칭되는 로그만 표시
- [ ] "데이터베이스" 단어 하이라이트
- [ ] 통계 업데이트됨

## 5️⃣ 로그 필터 (1분)
1. 레벨 필터에서 "ERROR" 선택
- [ ] ERROR 로그만 표시
- [ ] 빨간색 배지 확인
- [ ] 통계에서 ERROR 개수 확인

## 6️⃣ 새로고침 (30초)
1. 🔄 버튼 클릭
- [ ] "새로고침 완료" 알림 표시
- [ ] 3초 후 알림 자동 사라짐
- [ ] 데이터 갱신됨

## 7️⃣ 반응형 테스트 (1분)
1. F12 → 디바이스 툴바 (Ctrl+Shift+M)
2. iPhone SE 선택
- [ ] 메트릭 카드 1열 배치
- [ ] 스크롤 정상 동작
- [ ] 터치 가능

## 8️⃣ 메모리 테스트 (30초)
1. F12 → Console 탭
2. 페이지 새로고침 (F5)
3. 즉시 닫기 (Ctrl+W)
- [ ] 콘솔에 "🧹 정리 작업 수행 중" 로그 확인
- [ ] 이벤트 리스너 정리 확인
